# Cài đặt

Hướng dẫn này bao gồm hướng dẫn cài đặt chi tiết, yêu cầu hệ thống và khắc phục các vấn đề cài đặt phổ biến.

## Yêu cầu Hệ thống

### Yêu cầu Tối thiểu

| Yêu cầu | Thông số kỹ thuật |
|-------------|---------------|
| **Phiên bản macOS** | macOS 14.0 (Sonoma) hoặc mới hơn |
| **Bộ xử lý** | Apple Silicon (M1/M2/M3/M4/M5) hoặc Intel x86_64 |
| **Bộ nhớ** | Tối thiểu 4 GB RAM |
| **Dung lượng** | ~200 MB dung lượng trống |

### Khuyến nghị

- **macOS**: Phiên bản mới nhất để có hiệu suất tốt nhất
- **Bộ nhớ**: 8 GB RAM hoặc nhiều hơn cho cơ sở dữ liệu lớn
- **Dung lượng**: SSD để có hiệu suất tối ưu

<Note>
TablePro được xây dựng bằng các framework gốc của Apple và không yêu cầu bất kỳ môi trường runtime bổ sung nào như Java hay .NET.
</Note>

## Cài đặt qua Homebrew

Cách nhanh nhất để cài đặt TablePro là dùng [Homebrew](https://brew.sh):

```bash
brew install datlechin/tap/tablepro
```

<Tip>
Công thức cask tự động xóa thuộc tính quarantine của macOS sau khi cài đặt, nên bạn không cần chạy `xattr -cr` khi khởi chạy lần đầu.
</Tip>

Để cập nhật:

```bash
brew upgrade tablepro
```

Để gỡ cài đặt:

```bash
brew uninstall tablepro
```

Để xóa cả dữ liệu ứng dụng:

```bash
brew zap tablepro
```

## Tải xuống

### Từ GitHub Releases

1. Truy cập trang [TablePro Releases](https://github.com/datlechin/tablepro/releases)
2. Tìm bản phát hành mới nhất
3. Tải xuống DMG phù hợp cho Mac của bạn:

| Kiến trúc | File | Dành cho Mac với |
|--------------|------|---------------|
| Apple Silicon | `TablePro-arm64.dmg` | Chip M1, M2, M3, M4, M5 |
| Intel | `TablePro-x86_64.dmg` | Bộ xử lý Intel |

### Kiểm tra Kiến trúc Mac của bạn

1. Nhấp vào **menu Apple** () trên thanh menu
2. Chọn **About This Mac**
3. Tìm:
   - **Chip**: Apple M1/M2/M3/M4/M5 = Apple Silicon
   - **Processor**: Intel = x86_64

## Các bước Cài đặt

### Cài đặt Tiêu chuẩn

1. **Mở file DMG**
   - Nhấp đúp vào file `.dmg` đã tải xuống
   - Một cửa sổ Finder mới sẽ xuất hiện

2. **Kéo vào Applications**
   - Kéo biểu tượng TablePro vào shortcut thư mục Applications
   - Đợi quá trình sao chép hoàn tất

3. **Đẩy DMG ra**

   - Chọn **Eject**

4. **Khởi chạy TablePro**
   - Mở **Finder** > **Applications**
   - Nhấp đúp vào **TablePro**

### Bảo mật Khởi chạy Lần đầu

Vì TablePro không được ký với chứng chỉ Apple Developer, macOS sẽ hiển thị lỗi **"TablePro is damaged and can't be opened"** khi khởi chạy lần đầu. Điều này là bình thường — bạn cần xóa thuộc tính quarantine mà macOS thêm vào các ứng dụng tải xuống.

<Warning>
Đây là hành vi bình thường đối với các ứng dụng không được ký tải xuống từ internet. TablePro an toàn để sử dụng — mã nguồn hoàn toàn mở và có thể kiểm tra.
</Warning>

#### Xóa thuộc tính quarantine

Mở **Terminal** và chạy:

```bash
xattr -cr /Applications/TablePro.app
```

Sau đó mở TablePro bình thường từ Applications.

<Tip>
Bạn chỉ cần làm điều này một lần. Sau khi xóa thuộc tính quarantine, TablePro sẽ mở bình thường trong tất cả các lần khởi chạy sau.
</Tip>

#### Thay thế: System Settings

Nếu bạn vẫn thấy cảnh báo bảo mật sau khi chạy lệnh trên:

1. Mở **System Settings** > **Privacy & Security**
2. Cuộn xuống phần **Security**
3. Bạn sẽ thấy thông báo về TablePro bị chặn
4. Nhấp **Open Anyway**
5. Nhập mật khẩu nếu được yêu cầu

## Cập nhật TablePro

### Cập nhật Tự động

TablePro tích hợp tính năng cập nhật tự động được hỗ trợ bởi [Sparkle](https://sparkle-project.org/). Ứng dụng định kỳ kiểm tra phiên bản mới trong nền và thông báo cho bạn khi có bản cập nhật.

### Kiểm tra Cập nhật Thủ công

Để kiểm tra cập nhật thủ công:

1. Mở menu **TablePro** trên thanh menu
2. Nhấp **Check for Updates...**

### Cài đặt Cập nhật

Bạn có thể cấu hình hành vi cập nhật trong **Settings** > **General** > **Software Update**:

- Bật hoặc tắt kiểm tra cập nhật tự động
- Chọn tần suất kiểm tra cập nhật

<Tip>
Các kết nối và cài đặt của bạn được lưu trữ riêng biệt với ứng dụng và sẽ được giữ nguyên khi cập nhật.
</Tip>

### Vị trí Lưu trữ Dữ liệu

TablePro lưu trữ dữ liệu ở các vị trí sau:

| Dữ liệu | Vị trí | Loại lưu trữ |
|------|----------|--------------|
| **Kết nối** | `~/Library/Preferences/com.TablePro.plist` | UserDefaults |
| **Cài đặt** | `~/Library/Preferences/com.TablePro.plist` | UserDefaults |
| **Mật khẩu** | macOS Keychain (lưu trữ an toàn) | Keychain |
| **Lịch sử truy vấn** | `~/Library/Application Support/TablePro/query_history.db` | SQLite |

## Gỡ cài đặt

### Xóa Ứng dụng

1. Thoát TablePro nếu đang chạy
2. Mở **Finder** > **Applications**
3. Kéo **TablePro** vào Thùng rác

### Xóa Dữ liệu Ứng dụng (Tùy chọn)

Để xóa hoàn toàn tất cả dữ liệu TablePro:

```bash
# Xóa dữ liệu hỗ trợ ứng dụng
rm -rf ~/Library/Application\ Support/TablePro

# Xóa preferences
rm ~/Library/Preferences/com.TablePro.plist

# Xóa các mục Keychain (thông qua ứng dụng Keychain Access)
# Tìm kiếm "TablePro" và xóa các mục liên quan
```

<Warning>
Xóa dữ liệu ứng dụng sẽ xóa tất cả các kết nối đã lưu và lịch sử truy vấn của bạn. Đảm bảo xuất bất kỳ dữ liệu quan trọng nào trước.
</Warning>

## Khắc phục Sự cố

### "TablePro is damaged and can't be opened"

Làm theo hướng dẫn [Bảo mật Khởi chạy Lần đầu](#bảo-mật-khởi-chạy-lần-đầu) ở trên để xóa thuộc tính quarantine của macOS.

### "TablePro can't be opened because it is from an unidentified developer"

Làm theo hướng dẫn [Bảo mật Khởi chạy Lần đầu](#bảo-mật-khởi-chạy-lần-đầu) ở trên.

### Ứng dụng crash khi khởi chạy

1. Kiểm tra xem bạn đã tải xuống phiên bản kiến trúc đúng chưa
2. Đảm bảo bạn đang chạy macOS 14.0 hoặc mới hơn
3. Thử xóa và cài đặt lại ứng dụng
4. Kiểm tra Console.app để xem crash logs

### Vấn đề kết nối sau khi cài đặt

Nếu TablePro mở nhưng không thể kết nối với cơ sở dữ liệu:

1. Xác minh máy chủ cơ sở dữ liệu của bạn đang chạy
2. Kiểm tra cài đặt tường lửa (System Settings > Network > Firewall)
3. Đảm bảo máy chủ cơ sở dữ liệu cho phép kết nối từ localhost

## Build từ Source

Đối với các nhà phát triển muốn build TablePro từ source:

1. Clone repository:
```bash
git clone https://github.com/datlechin/tablepro.git
cd tablepro
```

2. Mở project Xcode:
```bash
open TablePro.xcodeproj
```

3. Build và chạy:
   - Chọn scheme **TablePro**
   - Nhấn `Cmd+R` để build và chạy

Xem hướng dẫn [Development Setup](/vi/development/setup) để biết hướng dẫn chi tiết.
