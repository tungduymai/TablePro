# Chào mừng đến với TablePro

TablePro là ứng dụng quản lý cơ sở dữ liệu macOS gốc được thiết kế để nhanh và nhẹ, với hỗ trợ SQL bằng AI được tích hợp sẵn. Được xây dựng hoàn toàn bằng SwiftUI và AppKit, ứng dụng ưu tiên các framework gốc của Apple và thư viện hệ thống để đạt hiệu suất tối ưu và trải nghiệm macOS liền mạch.

## Tính năng chính

### Trình soạn thảo SQL
Trình soạn thảo SQL mạnh mẽ với tô sáng cú pháp, số dòng và tự động hoàn thành thông minh thích ứng với schema cơ sở dữ liệu của bạn.

<Tip>
Sử dụng `Cmd+Enter` để thực thi câu truy vấn hiện tại hoặc văn bản đã chọn.
</Tip>

### Bảng dữ liệu
Xem và chỉnh sửa dữ liệu của bạn trong bảng hiệu suất cao, giống bảng tính với hỗ trợ chỉnh sửa nội tuyến, sắp xếp và lọc.

### Cấu trúc bảng
Kiểm tra và sửa đổi cấu trúc bảng, bao gồm cột, chỉ mục và khóa ngoại, tất cả từ giao diện trực quan.

### Nhập & Xuất
Nhập dữ liệu từ CSV, JSON và file SQL. Xuất dữ liệu của bạn ở nhiều định dạng để sao lưu hoặc di chuyển.

### Lịch sử truy vấn
Truy cập lịch sử truy vấn đầy đủ với tìm kiếm toàn văn. Nhanh chóng chạy lại hoặc sửa đổi các truy vấn trước đó.

### AI SQL Assistant
Trò chuyện với AI bằng ngôn ngữ tự nhiên để viết truy vấn, nhận gợi ý nội tuyến kiểu Copilot và click chuột phải để giải thích hoặc tối ưu SQL. Hỗ trợ Claude, OpenAI, Gemini, OpenRouter, Ollama và endpoint tùy chỉnh.

## Cơ sở dữ liệu được hỗ trợ

| Cơ sở dữ liệu | Trạng thái | Cổng mặc định |
|----------|--------|--------------|
| MySQL | Hỗ trợ đầy đủ | 3306 |
| MariaDB | Hỗ trợ đầy đủ | 3306 |
| PostgreSQL | Hỗ trợ đầy đủ | 5432 |
| SQLite | Hỗ trợ đầy đủ | Không áp dụng (dựa trên file) |
| MongoDB | Hỗ trợ đầy đủ | 27017 |

## Yêu cầu hệ thống

- **macOS**: 14.0 (Sonoma) trở lên
- **Kiến trúc**: Apple Silicon (arm64) hoặc Intel (x86_64)
- **Dung lượng**: ~50 MB cho ứng dụng
